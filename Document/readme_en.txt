2. How to build project file.

	For building, you need to install Xcode that is integrated development environment (IDE) of the Apple Computer Inc. 
	Please install Xcode from the attachment in installer disk of Mac OS X or ADC(Apple Developper Connection) site. 

	And you need "GData.framework" that is provided from Google Inc.
	Get project file from following address.
	Address : http://code.google.com/p/gdata-objectivec-client/
	You need to make "GData.framework" from this project file, and build it into "MacTubes" project file.

	- Open "MacTubes.xcodeproj" file with Xcode that exists in folder "MacTubes". 
	- Select menu "Project" -> "Set Active Build Configuration" -> "Release" in Xcode .
	- Click "Build" icon in upper part of windowB
	- Application file is created in folder "MacTubes" -> "build" -> "Release".
